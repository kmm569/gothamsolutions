What this template does

This spreadsheet is an automated reminder system designed to help you keep track of recurring and one-time tasks in a simple, organized way.

Once your tasks are entered, the sheet automatically checks which reminders are due each day and sends a summary email directly to your inbox. A built-in dashboard also provides a quick overview of active tasks and upcoming reminders.

Key features

  • Daily reminder emails automatically sent from the spreadsheet
  • Support for both one-time reminders and repeating schedules
  • Simple task list with notes, start dates, and reminder settings
  • Built-in dashboard showing active tasks and upcoming reminders
  • Customizable email subject and message templates
  • Runs automatically once configured using Google Apps Script

Ways you can use it

  • Keeping track of recurring household tasks and chores
  • Reminders for bills, subscriptions, and important payments
  • Managing small business routines like invoices or client follow-ups
  • Tracking recurring service tasks like landscaping visits, maintenance checks, or routine inspections
  • Maintaining personal routines such as workouts or medication schedules
  • Handling recurring work processes like reports or content publishing

How it works

  1. Enter tasks, details, and start dates on the Reminders sheet
  2. Select whether the task is one-time or recurring
  3. Set the reminder interval and mark the task as Active
  4. The system automatically checks which reminders are due each day
  5. You receive a daily email summarizing tasks that require attention
  6. The sheet updates the next reminder date automatically
